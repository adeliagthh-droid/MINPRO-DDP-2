# PENJELASAN
1. Setelah program dijalankan, pengguna akan langsung diarahkan ke halaman login.
   
Di sini, pengguna diminta untuk memasukkan username dan password.
Jika data yang dimasukkan benar, program akan menampilkan pesan sambutan sesuai dengan role pengguna (Manager atau Karyawan).
Jika data yang dimasukkan salah, program akan memberikan peringatan "Username atau password salah!" dan pengguna diminta login kembali.
<img width="1636" height="657" alt="Screenshot 2025-09-28 215014" src="https://github.com/user-attachments/assets/f352c8e3-4b3d-4231-a67a-9e69cb773e98" />  

2. MENU MANAGER
   
Bagi pengguna dengan role Manager, setelah login akan ditampilkan Menu Utama Manager yang terdiri dari lima pilihan  

- Lihat Travel Log
   
Jika daftar perjalanan kosong, program akan menampilkan pesan: "Belum ada catatan perjalanan."
Jika ada data, program akan menampilkan daftar perjalanan dengan format rapi, mencakup nomor perjalanan, destinasi, tanggal, dan catatan.
<img width="654" height="271" alt="Screenshot 2025-09-28 220112" src="https://github.com/user-attachments/assets/e6258a30-b31e-483e-80b3-9baf6f61246d" />
<img width="407" height="196" alt="Screenshot 2025-09-28 220137" src="https://github.com/user-attachments/assets/54134018-1bb1-427d-b5e0-a22cf8509e61" />

- Tambah Travel Log
  
Program akan meminta pengguna memasukkan data perjalanan baru, yaitu:
Tujuan perjalanan
Tanggal perjalanan
Catatan tambahan
Setelah semua data dimasukkan, program akan menyimpan perjalanan tersebut dan memberikan konfirmasi berupa pesan: "Catatan perjalanan berhasil ditambahkan!"
Dengan cara ini, pengguna dapat menambahkan perjalanan sebanyak yang diperlukan.  
<img width="575" height="256" alt="Screenshot 2025-09-28 221356" src="https://github.com/user-attachments/assets/8a437c08-68e9-40e5-afd8-267b8b01e94c" />  

- Update Travel Log
  
Program menampilkan daftar perjalanan.
Pengguna diminta memilih ID perjalanan yang ingin diperbarui.
Setelah itu, pengguna dapat mengganti tujuan, tanggal, atau catatan sesuai kebutuhan.
Jika ID yang dimasukkan tidak ditemukan, program akan menampilkan pesan: "ID tidak ditemukan!".

- Hapus Travel Log
  
Program menampilkan daftar perjalanan.
Pengguna diminta memilih ID perjalanan yang ingin dihapus.
Jika ID ditemukan, data perjalanan akan dihapus dan program menampilkan pesan: "Catatan perjalanan berhasil dihapus!".
Jika ID salah, program akan menampilkan pesan kesalahan yang ramah.

- Logout
  
Jika pengguna memilih opsi ini, program akan mengakhiri sesi Manager dan kembali ke halaman login.
